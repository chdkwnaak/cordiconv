GPS 좌표 변환기

단일 좌표를 바로 변환하거나 txt, csv, xlsx, xls 파일을 불러와 대량 변환할 수 있습니다.

사용자 지정 Proj4 직접 입력

좌표계 목록에서 사용자 지정을 선택한 쪽의 Proj4 정의를 입력합니다.

파일을 드래그 앤 드롭하거나 클릭해서 선택

지원 형식: .txt, .csv, .xlsx, .xls

지도 미리보기

사용방법

좌표계 선택 후 단일 좌표 또는 여러 좌표를 변환합니다.

1. 좌표계 선택

상단에서 원본 좌표계와 변환할 좌표계를 먼저 선택합니다. 목록에 없는 좌표계는 사용자 지정 Proj4 직접 입력을 선택한 뒤 Proj4 정의를 입력합니다.

2. 단일 좌표 변환

  1. 단일 1번째 값에는 경도/Easting, 단일 2번째 값에는 위도/Northing을 입력합니다.
  2. 단일 변환 버튼을 누르거나 입력칸에서 Enter를 누릅니다.
  3. 단일 변환 결과와 오른쪽 결과 영역에서 변환값을 확인합니다.

3. 대량 좌표 변환

  1. 좌표를 한 줄에 하나씩 입력합니다. 예: 126.9784147 37.5666805
  2. txt, csv, xlsx, xls 파일은 드래그하거나 클릭해서 불러올 수 있습니다.
  3. csv, xlsx, xls 파일은 1번째/2번째 좌표 열과 첫 행 헤더 사용 여부를 선택한 뒤 변환하기를 누릅니다.

4. 축 순서

일반 입력 순서는 경도/Easting, 위도/Northing입니다. 원본 자료가 위도/Northing, 경도/Easting 순서라면 입력 좌표 순서 바꾸기를 체크합니다.

5. 결과 사용

결과는 클립보드로 복사하거나 CSV 파일로 다운로드할 수 있습니다.
